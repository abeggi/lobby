LOBBY

Un frontend FastAPI proxy per navigare la libreria del tuo server Jellyfin. Leggero, scuro, senza fronzoli.

ℹ️ Codice su GitHub Funzionalità

Funzionalità

Tutto quello che serve per esplorare la tua mediateca, niente di più.

🎬

Film e Serie

Naviga la libreria completa del tuo Jellyfin con pagine dedicate per film e serie TV.

🔍

Filtri avanzati

Filtra per anno, ordina per data e cerca per titolo. Trovi quello che cerchi in un click.

📄

Pagina dettaglio

Poster, backdrop, logo, cast, link a trailer, TMDB, TVDB e IMDb. Tutto in una pagina.

🖼️

Proxy immagini

Le immagini passano dal backend: la API key di Jellyfin resta nascosta dal browser. Cache 24h inclusa.

📱

Responsive

Layout fluido con sidebar collassabile. Funziona su desktop, tablet e telefono.

🌙

Tema scuro

Accento indaco su sfondo gray-900. Solo dark mode, senza compromessi.

Stack tecnico

Backend Python, frontend vanilla. Nessun framework inutile.

  • Backend: FastAPI
  • HTTP: httpx
  • Server: uvicorn
  • Frontend: Vanilla HTML/CSS/JS
  • Python: 3.12+
  • Cache: LocalStorage (60min)
  • Proxy: Immagini con cache 24h

Avvio rapido

Clona, configura, avvia. In un minuto sei online.

$ git clone https://github.com/abeggi/lobby
$ cd lobby
$ python3 -m venv .venv
$ .venv/bin/pip install -r requirements.txt
$ cp .env.sample .env
# modifica .env con i tuoi dati Jellyfin
$ .venv/bin/uvicorn main:app --host 0.0.0.0 --port 8080
